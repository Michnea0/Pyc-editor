# Pyc-editor
Un code editor simplu si curat in python cu tkinter ca gui si cateva libs in plus

## Instalare
Pentru a rula acest cod editor este nevoie de python si cateva bibloteci

```python
   pip install tkinter
```
```python
   pip install colorama
```
```python
   pip install json
```
